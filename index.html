About Me

Halo! Saya adalah seorang pengembang web dengan minat besar dalam menciptakan pengalaman digital yang menarik, interaktif, dan fungsional. Dengan latar belakang dalam desain antarmuka pengguna (UI) dan pengembangan front-end, saya selalu berusaha menggabungkan estetika visual dengan performa yang optimal.

Saya memulai perjalanan di dunia teknologi sejak memasuki dunia perkuliahan dan sejak itu terus mengasah keterampilan saya di berbagai proyek, mulai dari situs web portofolio pribadi hingga aplikasi web kompleks berskala besar. Keahlian saya mencakup HTML, CSS, JavaScript, dan berbagai framework seperti React dan Vue, serta pengalaman dalam integrasi API dan optimasi performa web.

Bagi saya, coding bukan hanya sekadar menulis baris demi baris kode, tapi juga tentang memecahkan masalah, berpikir kreatif, dan menciptakan solusi digital yang memberikan nilai nyata bagi pengguna. Saya juga percaya pada pentingnya desain responsif agar website dapat diakses dengan baik di berbagai perangkat.

Di luar dunia coding, saya senang mengeksplorasi tren teknologi terbaru, mengikuti komunitas developer, dan sesekali berbagi ilmu lewat blog atau workshop. Saat ini, saya terbuka untuk peluang kolaborasi, freelance, atau kerja penuh waktu di bidang pengembangan web.
